Step 1: SQL Server <> Python

November 5, 2020

The first step involved downloading the training set, which will be done monthly. The testing set will be downloaded hourly or by user request. Below a guide to achieving this.

  • DOWNLOADING training set (monthly)
    • Required files:
      dataset1
dataset2
dataset3
    • Libraries required 
      import pyodbc  
import pandas as pd
    • Accesing SQL server 
      with open(r'C:\Users\path to where your connection credential are.txt') as file:
    SQL_credentials = file.read().replace('\n', '')
      conn = pyodbc.connect(SQL_credentials)
    • Querring variables from SQL
      sql_query = pd.read_sql_query('''SELECT [Quoting].[dbo].[DavidSoanes_A_Distributor].Name as [Quote Number],
                                [Quoting].[dbo].[DavidSoanes_A_Distributor].QuoteVersion,
                                [Quoting].[dbo].[DavidSoanes_A_Distributor].TSICode as PART,
                                [Quoting].[dbo].[DavidSoanes_A_Distributor].ItemDescription,
                                [Quoting].[dbo].[DavidSoanes_A_Distributor].ProductDescription as Product,
                                SUM([Quoting].[dbo].[DavidSoanes_A_Distributor].Quantiry) as [Claim Qty],
                                [Quoting].[dbo].[DavidSoanes_A_Distributor].DateCreated,
                                [Quoting].[dbo].[DavidSoanes_A_Distributor].Stage,
                                SUM([Quoting].[dbo].[DavidSoanes_A_Distributor].StdXqty) as [Cost Of Sales],
                                SUM([Quoting].[dbo].[DavidSoanes_A_Distributor].Margin) as Margin,
                                SUM([Quoting].[dbo].[DavidSoanes_A_Distributor].ContractorNet) as [Contractor Net],
                                [Quoting].[dbo].[DavidSoanes_A_Distributor].ProductRange as Brand,
                                [Quoting].[dbo].[DavidSoanes_A_Distributor].sfdcAccountID as [AccountID%]
                                
FROM [Quoting].[dbo].[DavidSoanes_A_Distributor]
WHERE [Quoting].[dbo].[DavidSoanes_A_Distributor].DateCreated >= DATEADD(yy, -3, GETDATE())
GROUP BY 
[Quoting].[dbo].[DavidSoanes_A_Distributor].TSICode,
                                [Quoting].[dbo].[DavidSoanes_A_Distributor].Name,
                                [Quoting].[dbo].[DavidSoanes_A_Distributor].Product,
                                [Quoting].[dbo].[DavidSoanes_A_Distributor].DateCreated,
                                [Quoting].[dbo].[DavidSoanes_A_Distributor].Stage,
                                [Quoting].[dbo].[DavidSoanes_A_Distributor].ProductRange,
                                [Quoting].[dbo].[DavidSoanes_A_Distributor].sfdcAccountID,
                                [Quoting].[dbo].[DavidSoanes_A_Distributor].QuoteVersion,
                                [Quoting].[dbo].[DavidSoanes_A_Distributor].ItemDescription,
                                [Quoting].[dbo].[DavidSoanes_A_Distributor].ProductDescription
ORDER BY ([Quoting].[dbo].[DavidSoanes_A_Distributor].DateCreated);
''',conn) 
df = pd.DataFrame(sql_query)
df.to_csv (r'C:\Users\path to where your file is.csv', index = False)
